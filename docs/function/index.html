    public async signatureGenerate(payload: string, action: Object): string source 

    import {signatureGenerate} from 'students-api/students-api/services/signatureService.js'

    Generate a random signature and add it to redis signed urls allowlist with the payload value

    Params:

    NameTypeAttributeDescription
    payload string

    The value to save on redis
    action Object

    The action to save with payload
    action.id string
    • optional
    • default: 'GENERIC'

    The id (type) of action
    action.expiration.time number
    • optional
    • default: 5

    The expiration time value
    action.expiration.time string
    • optional
    • default: 'd'

    The expiration time unit

    Return:

    string

    signature

    Example:

    const payload = user.id
const action = { id: 'EMAIL_VALIDATION', expiration: { time: 5, unit: 'd' }}
const signature = await signatureGenerate(payload, action)
